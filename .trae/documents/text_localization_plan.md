# 文本汉化实现计划 - 通过映射表替换文本

## 项目概述

本计划旨在通过创建一个文本映射表，在渲染层面实现硬编码文本的汉化，而不需要修改原始英文文本代码。

## 技术实现细节

### 映射文件格式

使用 JSON 格式存储映射，示例：

```json
{
  "You have died.": "你已经阵亡了。",
  "No Pilot Loaded": "没有加载任何驾驶员",
  "Commodity": "商品",
  "Price": "价格"
}
```

### 实现架构

1. 创建 `TextMapper` 类，负责加载和管理映射表
2. 在 `Font` 类中集成 `TextMapper` 功能
3. 在文本渲染前进行映射替换
4. 确保所有文本渲染路径都经过映射处理

### 关键修改点

* `Font.cpp`: 修改 `Draw` 方法，添加文本映射逻辑

* 新增 `TextMapper.cpp` 和 `TextMapper.h`: 实现映射表管理

* 新增 `text_mappings.json`: 存储文本映射

## 风险评估

1. **性能风险**: 文本映射可能会影响渲染性能，需要优化查找算法
2. **兼容性风险**: 映射表可能与游戏更新不兼容，需要定期维护
3. **布局风险**: 中文文本长度可能与英文不同，可能导致布局问题

## 预期成果

* 游戏中所有硬编码文本都显示为中文

* 无需修改原始英文代码，便于后续游戏更新

* 映射表可独立维护，便于翻译更新
